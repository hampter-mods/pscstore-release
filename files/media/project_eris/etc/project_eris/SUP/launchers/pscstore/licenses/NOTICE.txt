PSC Store Third-Party Notices
=============================

This directory contains third-party license texts, notices, and source
availability notes for components used by or distributed with PSC Store.

Bundled or distributed components include:

- RetroArch 1.8.8 patched build
- PCSX-ReARMed disc-project core
- Alex Free libcrypt-patcher / lcp
- 7-Zip / 7zr 23.01
- curl / libcurl 8.5.0
- OpenSSL 1.1.1u
- zlib compression support
- curl CA extract / Mozilla CA bundle
- UPX decompressor stub in the packed RetroArch binary
- Linux kernel optical-drive modules: cdrom.ko, sg.ko, sr_mod.ko
- Project Eris / ModMyClassic scripts and integration points
- PSC Store bundled pcsx_data.db, a lightly adjusted version of the pre existing Project
  Eris metadata database
- Game cover art based on materials already provided by Project Eris
- PSC Store app artwork, icons, and error-screen images generated by the
  project maintainer

Runtime/build dependencies referenced by PSC Store include:

- SDL2, SDL2_image, SDL2_ttf
- libpng
- SQLite / libsqlite3
- OpenGL ES / EGL

Project Eris function scripts use the existing Project Eris-provided sqlite3
command-line tool.

Font references:

- Cabin Regular
- Ezarion
- Rounded M+ 1c

The listed fonts are referenced as Project Eris/theme-provided OFL fonts.
PSC Store does not bundle those font files in this package.

Background shader reference:

- "fill glow & outline glow" by Padej_ on Shadertoy

See Shadertoy-fill-glow-outline-glow.txt for the attribution note for the shader-derived background cube.

Binary provenance notes:

- The Linux optical-drive modules were sourced from honeylab's "CD-ROM driver
  On Playstation Classic" article:
  https://honeylab.hatenablog.jp/entry/2019/01/16/002111
